Compilation of R codes associated with Derville et al. 2024 A predictive krill distribution model for Euphausia pacifica and Thysanoessa spinifera using scaled acoustic backscatter in the Northern California Current. Progress in Oceanography.

These codes include all models tested as part of the above manuscript.
For a streamlined reproducible version including only the models selected as "optimal" in the manuscript, check out repository "Selected_models".
